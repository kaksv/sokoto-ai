# SokoAgent AI — Frontend

This is the frontend for [SokoAgent AI](../README.md), an AI-powered USSD platform for Ugandan smallholder farmers.

## Quick Start

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`.

## Build

```bash
npm run build
```

Output in `dist/` — deploy as a static site on Render.

## Tech Stack

React 18 + Vite • Tailwind CSS v3 • Framer Motion • Lucide React • React Router DOM v6

See the [root README](../README.md) for full project documentation.
